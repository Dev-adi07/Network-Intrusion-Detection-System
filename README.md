# 🛡️ Network Intrusion Detection System

A Machine Learning based Intrusion Detection System (IDS) designed to detect and classify malicious network traffic using advanced classification algorithms.

## 🚀 Features

- Data Preprocessing & Feature Engineering
- Decision Tree Classification
- Random Forest Classification
- XGBoost Classification
- Hyperparameter Optimization
- Network Attack Detection
- Web-Based Prediction Interface

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Flask
- Joblib

## 📂 Project Structure

- app.py → Flask Web Application
- train_model.py → Model Training
- model.pkl → Trained Model
- scaler.pkl → Feature Scaler
- encoders.joblib → Label Encoders
- Intrusion.Detection.ipynb → Complete Notebook

## 🎯 Objective

To identify malicious network activities and improve cybersecurity through Machine Learning based intrusion detection.
